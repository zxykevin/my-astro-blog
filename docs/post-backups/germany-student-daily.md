# 德国留学日常

德国留学最真实的样子，往往不是朋友圈里那张滤镜很重的街景，而是早晨赶车、课间买咖啡、晚上对着德语邮件反复确认附件有没有传对。

## 上午：从电车开始的一天

如果当天有早课，我会提前看好 DB 或本地交通 App。德国公共交通整体方便，但临时延误也很常见，所以出门前多留十分钟会安心很多。

到学校后，第一件事通常是找一个能充电的位置。课堂节奏比想象中更强调主动提问和课前阅读，老师不会把所有知识点都拆到很细，很多理解要靠自己课后补。

## 中午：食堂和超市

学生食堂是控制预算的好地方。想换口味时，我会去亚超补一点调料，简单做米饭、咖喱、拉面或者炒菜。真正开始留学后才发现，会做几道稳定的家常菜是一种很强的生活技能。

## 下午：学习、手续和邮件

德国生活里有很多“纸面流程”：住房合同、保险、银行、居留、广播电视费。每一项看起来都不复杂，但如果德语邮件读得慢，就会很消耗精力。

我的经验是：

- 重要文件按主题建文件夹保存。
- 每次办手续都记录日期、联系人和下一步。
- 不确定的地方尽早发邮件确认。

## 晚上：给自己一点缓冲

晚上会留一点时间看番、整理照片，或者写博客。海外生活不总是浪漫的，但把每天的小事写下来，会让许多当下觉得麻烦的瞬间，变成以后能回头看的轨迹。
